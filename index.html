Resumo básico JSON


Estrutura básica de JSON:

Objetos JSON = Formados por um ou mais pares de valor-chaves.

Devem ser agrupados dentro de chaves "{ }" e os pares devem ser separados por vírgula ",".


Par valor-chave = Funcionam como um dicionário, dois dados separados por 2 pontos ":", podendo ser strings, intenger ou boolean, onde o primeiro valor é a "chave" e o segundo o "valor" do par.

Objetos JSON podem ser utilizados como um valor par para uma chave.


Exemplo:

{"2022": {"ATIVO": 100, "PASSIVO": 70, "PL":30, "DRE":20}}


Matriz (Array) JSON = Um conjunto de dados que podem ser strings, intenger, etc simples ou um próprio objeto JSON.

Devem ser agrupados dentro de colchetes "[ ]" e separados por vírgula ",".


Exemplo:

[{"2022": {"ATIVO": 100, "PASSIVO": 70, "PL":30, "DRE":20}}, {"2023": {"ATIVO": 120, "PASSIVO": 70, "PL":50, "DRE":30}}]


Uma Array também pode ser utilizada como valor para um par valor-chave dentro de um Objeto JSON:

{"EmpresaExemplo S.A" : [{"2022": {"ATIVO": 100, "PASSIVO": 70, "PL":30, "DRE":20}}, {"2023": {"ATIVO": 120, "PASSIVO": 70, "PL":50, "DRE":30}}]}
